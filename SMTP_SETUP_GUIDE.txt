Gmail（Googleアカウント）を使用してメールを送信するための設定手順です。

1. **Googleアカウントの準備**
   Gmailを外部アプリ（今回の予約システム）から利用するには、「アプリパスワード」の生成が必要です。
   通常のログインパスワードは使用できません。

2. **アプリパスワードの生成手順**
   1. Googleアカウント管理ページ（https://myaccount.google.com/）にアクセスします。
   2. 左メニューの「セキュリティ」をクリックします。
   3. 「Google へのログイン」セクションにある「2 段階認証プロセス」が「オン」になっていることを確認します。（オフの場合はオンにしてください）
   4. 「2 段階認証プロセス」をクリックし、ページ下部の「アプリ パスワード」を選択します。
      （見つからない場合は、上部の検索バーで「アプリ パスワード」と検索してください）
   5. 「アプリ名」に適当な名前（例: GuesthouseBookingSystem）を入力し、「作成」をクリックします。
   6. 16桁のパスワードが表示されます（例: `xxxx xxxx xxxx xxxx`）。これが「アプリパスワード」です。この画面を閉じずに次の手順へ進んでください。

3. **設定ファイルの更新**
   サーバー上の `includes/mail_config.php` をテキストエディタで開き、以下の項目を書き換えてください。

   ```php
   // 認証情報 (※必ず書き換えてください)
   define('SMTP_USERNAME', 'あなたのメールアドレス@gmail.com');
   define('SMTP_PASSWORD', 'さきほど取得した16桁のアプリパスワード'); // スペースはあってもなくても構いません
   ```

   また、送信元アドレスも必要に応じて変更してください。
   ```php
   define('MAIL_FROM_ADDRESS', 'あなたのメールアドレス@gmail.com');
   ```

4. **確認**
   設定後、予約テストを行い、メールが届くか確認してください。
   もし届かない場合は、ウェブサーバーのエラーログを確認してください。
