# Guesthouse Marusho Reservation System

ゲストハウス丸正の予約管理システムです。

## 機能一覧

### ユーザー向け機能
- **空室検索・予約**: 日付と部屋タイプを指定して予約が可能。
- **予約確認**: 予約完了時に発行されるトークンURL、またはログイン後のマイページで予約を確認。
- **多言語対応**: 日本語と英語の切り替えが可能。

### 管理者向け機能
- **ダッシュボード**: 予約状況の概要を確認。
- **予約管理**: 予約の作成、編集、キャンセル、検索。
- **空室カレンダー**: カレンダー形式で予約状況を視覚的に管理。
- **顧客管理**: 顧客情報の編集、削除、パスワードリセット、予約履歴の確認。
- **部屋・部屋タイプ管理**: 部屋情報の追加・編集。

## 最近の実装機能

### セキュリティ強化
- **予約トークン**: ゲスト予約時に推測不可能なトークンを発行し、URLによる第三者の不正アクセスを防止。
- **パスワード管理**: 管理者がユーザーのパスワードを安全に変更できる機能を追加。

### UX向上
- **リアルタイム空室チェック**: 管理画面での予約作成時、日付・部屋選択時にリアルタイムで空室状況を判定し、ダブルブッキングを警告。
- **モバイル対応**: 管理画面をスマートフォンやタブレットでも操作しやすいレスポンシブデザインに改修。

## インストール・セットアップ

1. データベースの作成と初期化
   ```bash
   mysql -u root -p < database.sql
   ```
2. データベース接続設定
   `includes/db_connect.php` の設定を環境に合わせて変更してください。
